Compressor Run Capacitor Wiring

Webasto FCF16000 compressor controlled by Cruisair SMX II P/L board
Standard single-phase hermetic compressor with C / S / R terminals

What The Run Capacitor Does

The run capacitor creates the phase shift needed to keep the compressor motor's start winding energized during operation. It sits between the R (Run) and S (Start) terminals of the compressor. Without it, the compressor either won't start or will hum, draw locked rotor amps, trip the overload, and shut down.

Compressor Terminal Reference

Look at the top of the compressor — there's a terminal cover (usually held on by a clip or screw). Under it you'll find three spade terminals stamped or labeled:
Terminal Name What It Does
C Common Main power feed from P/L board (via TRIAC switch). This is the "always hot when running" terminal.
S Start Start winding. Run cap connects here.
R Run Run winding. Power feed AND run cap connect here.
How to identify if not labeled: Using a multimeter on the disconnected compressor, measure resistance between all three pairs. The pair with the highest resistance is S-to-R (start + run windings in series). The remaining terminal is C. The lower resistance from C is to R (run winding); the higher resistance from C is to S (start winding).

Wiring Summary — 4 Connections Total

# From To Notes
1 P/L board COMP output Compressor C (Common) Switched hot — this is the line the TRIAC controls
2 P/L board N2 (neutral) Compressor R (Run) Neutral return to run winding
3 Run cap terminal 1 Compressor S (Start) Cap not polarized, either terminal works
4 Run cap terminal 2 Compressor R (Run) Shares the R terminal with the neutral wire
🚨 KILL THE BREAKER FIRST. Then discharge the old capacitor by shorting its terminals together with an insulated screwdriver before touching anything. Caps hold charge even with power off.
⚠ IMPORTANT NOTES:

The run cap is NOT polarized — it doesn't matter which cap terminal goes to S vs R. Either way works.

The cap sits across S and R only. It does NOT connect to C. If you find a wire from the cap going to C, something's wrong.

On the Cruisair SXF5-16 schematic (Dwg C0010001), the P/L board's TRIAC switches the COMP output to the compressor's C terminal. The run capacitor and compressor motor are a self-contained subcircuit between C, S, and R — the P/L board doesn't directly interact with the cap.

If you also have a start capacitor and/or start relay: that's a separate hard-start circuit. The start cap is typically smaller and darker (often black), wired in parallel with the run cap but switched by a potential relay or PTC device. Not all units have one. If yours doesn't currently have one and the compressor is struggling to start, a Micro-Air EasyStart is the modern fix.

Testing The Old Capacitor

Before swapping: set your multimeter to capacitance mode (if it has one) and measure across the two cap terminals (disconnected from everything, discharged first).

A healthy 25 MFD cap should read somewhere between 22.5 and 27.5 MFD (within ~10%). If it reads significantly below 20 MFD, it's weak and the compressor is struggling. If it reads 0 or shows a short, it's dead.

No capacitance mode? Use ohms: touch probes to cap terminals. Should briefly show near-zero then climb steadily toward infinity (open). If it stays at zero = shorted. If it starts at infinity and never moves = open/dead.

Also visually inspect: bulging top/bottom, leaking fluid, or scorch marks = replace immediately.
