Webasto FCF16000 (115V) compressor/condenser → Cruisair SMX II P/L board (DX) control
Source: Cruisair L-2382, Dwg C0010001 (SXF5-16, 115V) & Webasto FCF Series Install Manual (Rev L, 8/2018)
✅ VERIFICATION RESULT: Sonnet's original schematic is fundamentally correct on all critical wiring paths. The Cruisair SMX II P/L board terminal layout, pressure switch rerouting procedure, pump terminal wiring, and signal cable routing all match the official documentation. Two minor clarifications added below.

1. Shore Power → Cruisair P/L Board → Webasto Compressor Unit CORRECT

Wire From To Terminal Notes
Line (Black) Shore panel breaker Cruisair P/L board L1 115V hot, 12 AWG min (ABYC)
Neutral (White) Shore panel neutral bus Cruisair P/L board N1 12 AWG min
Ground (Green) Shore panel ground bus Cruisair P/L board GRND stud ABYC E-8: MUST be landed
Comp. Line Cruisair P/L board Webasto FCF terminal strip P/L: COMP → FCF: L Switched by P/L board TRIAC
Comp. Neutral Cruisair P/L board Webasto FCF terminal strip P/L: N2 → FCF: N Common neutral return
How this works: Shore power feeds into the Cruisair P/L board at L1/N1. The P/L board uses a TRIAC to switch the compressor on/off based on thermostat demand. The switched output goes from the COMP terminal on the P/L board to the Webasto's L (line) terminal. The neutral passes through from N2 to the Webasto's N terminal. The Webasto's own digital display/controller is bypassed entirely — the Cruisair SMX II is now the brain.

2. Seawater Pump (March LC-3CP-MD, 115V) CORRECT

Wire From To Terminal Notes
Pump Line (Black) Cruisair P/L board March pump hot lead PUMP (1) P/L board switches pump on before compressor
Pump Neutral (White) Cruisair P/L board March pump neutral lead PUMP (2) Neutral return
Confirmed from both manuals: The Webasto FCF manual labels pump terminals as "Cooling Pump Line In" (terminal 1) and "Cooling Pump Neutral" (terminal 2). The Cruisair P/L board has matching PUMP terminals. In the Frankenstein config, the pump wires go to the Cruisair P/L board pump terminals (NOT the Webasto terminal strip), because the Cruisair board controls pump sequencing (pump starts a few seconds before compressor kicks on).

If only one AC unit uses this pump: no separate breaker needed — pump wires go straight to the P/L board terminals. If multiple AC units share one pump, a pump relay (Cruisair PR3X or PR8X) is required per the L-2382 manual.

3. Pressure Switch Rerouting CORRECT — THE KEY FRANKENSTEIN STEP

Switch Action P/L Board Terminal Notes
High Pressure Switch Disconnect from Webasto board; wire to Cruisair P/L HIGH PRESS (push-on terminals) NC switch, opens on high head pressure
Low Pressure Switch Disconnect from Webasto board; wire to Cruisair P/L LOW PRESS (push-on terminals) Optional per Cruisair; recommended
— Old Webasto switch wires: butt-splice together — Bypasses switches on Webasto side
⚠ THIS IS THE #1 GOTCHA ON FRANKENSTEIN INSTALLS.

The Cruisair L-2382 manual explicitly states: "A high-pressure switch must be connected to the SMX P/L board. It should be disconnected from its in-line application, and connected to the SMX P/L board at the push-on terminals labeled 'HIGH PRESS'. The wires that were connected to the high-pressure switch should then be butt connected together, effectively bypassing the switch."

If you skip this step, the Cruisair board has no fault protection and won't know if the system is over-pressuring. Meanwhile, the Webasto's own board (which you've effectively disconnected by using the Cruisair as the controller) may still try to act on the switches and fight the Cruisair logic.
Clarification vs. original schematic: The low pressure switch is described as "recommended but optional" in the Cruisair manual if the original unit didn't have one. The Webasto FCF16000 does have both HP and LP protection built into its own controller (error codes E1 for high press, E3 for low press per the FCF manual), so the physical switches on the refrigerant lines should be present. Reroute both to the Cruisair P/L board.

4. SMX II Signal Cables (Low Voltage DC) CORRECT

Cable From To Notes
CX cable (shielded, 3-conductor, 4-pin) P/L board CX jack SMX II AB keypad (cabin wall) Use CXP## cable if using SMXir keypad
TSEP cable (4-conductor, RJ-11) P/L board "INSIDE TSE" jack Temp sensor behind return air grille Sensor must NOT touch evaporator coil
🚨 CRITICAL: Route CX/CXP and TSEP cables AWAY from all 115V power cables.
Per L-2382: "Do not route these cables beside A/C power cables, high voltage wiring, or antenna wires." Interference causes erratic SMX display behavior and temperature read errors. Standard phone cable will NOT work — must be Cruisair CX/CXP cables.

5. Evaporator Blower (Fan) CLARIFICATION

This is where it gets unit-specific. The Webasto FCF16000 has a built-in 3-speed blower that was designed to be controlled by the Webasto's own digital controller. The Cruisair SMX II P/L board has blower speed control outputs on standard Cruisair SX/FX units (the TRIAC-controlled blower circuits visible on the SXF5-16 schematic, Dwg C0010001).

For the Frankenstein config: If the Webasto blower is wired through the Webasto control board, you have two options:

Option A (simpler): Run the blower directly on a fixed speed (wire blower power straight to L1/N, bypassing both controllers). Fan runs continuously whenever the breaker is on. This is the most common approach on Frankenstein installs.

Option B (full integration): Wire the Webasto blower motor leads into the Cruisair P/L board's fan output terminals. This gives you SMX-controlled variable speed. However, this requires matching the Webasto blower motor's wiring (capacitor values, speed taps) to what the Cruisair board expects — this is unit-specific and you should check the diagram on the inside of your P/L box door.

Sonnet's original schematic didn't explicitly address blower wiring, which is fine — this is the one area that depends on what's physically in front of you. Check which option your existing install uses.

7. Pre-Power Checklist

  • Green ground wire landed at GRND stud on P/L board (missing = electrolysis risk to thru-hulls)
  • Webasto pressure switch wires disconnected from Webasto board and rerouted to Cruisair HIGH PRESS / LOW PRESS terminals
  • Old Webasto pressure switch wires butt-spliced together (bypassed on Webasto side)
  • Seawater pump wired to Cruisair P/L board PUMP terminals (NOT Webasto terminals 1/2)
  • Webasto FCF pump terminals (1, 2) left disconnected / unused
  • CX/TSEP signal cables NOT bundled with 115V power cables
  • Seawater strainer clear, pump primed, raw water flow verified before first compressor start
  • P/L board 3.15A fuse present and intact
  • Breaker sized per Webasto data plate (30A HACR for 16K unit with 60A LRA)
  • Wait 30+ seconds between power cycles (compressor damage risk)

8. Quick Troubleshooting

SMX display won't light up?
→ Check CX/CXP cable at both ends (both plugs fully seated, all 4 pins engaged)
→ Verify 115V at L1-N1 terminals inside P/L box (open cover, use meter)
→ Check 3.15A fuse on P/L board
→ Factory memory reset: press OFF/SET/FAN together, then SET

``` Compressor runs but no cooling?
→ #1 cause: inadequate seawater flow. Check strainer, verify discharge stream overboard
→ High-pressure fault will shut down compressor (check SMX display for fault code)
→ Low refrigerant charge

Erratic temperature display?
→ CX/TSEP cable routed too close to 115V wiring (interference)
→ TSEP sensor touching evaporator coil (must be in return air path only)
→ Calibrate: press OFF, then SET, then SET+UP together — adjust to match actual temp

Compressor trips breaker on startup?
→ 25A breaker is undersized for 60A LRA. Need 30A HACR minimum
→ Or add EasyStart soft starter to reduce inrush to ~25A ```
